# Phase 7.5: Dashboard Weekly Training Plan Overview — ✅ Complete (PR #127)

## Goal

Add a "This Week" training plan overview card to the dashboard, positioned above the Training Load card. When a user has an active structured training plan, the dashboard should show what's planned for the current week at a glance.

**Discovered during:** OB-15 manual testing (user feedback: "can we include an overview of the current week training plan in the dashboard on top of the training load")
**Priority:** Medium - improves daily UX for users with structured plans

---

## Current State

- Dashboard shows 4 cards: Today's Workout, Training Load, Recent Activities, Upcoming Events
- Training plan screen (`apps/mobile/app/(tabs)/plan.tsx`) shows full plan with phases and weekly volume
- `getActiveTrainingPlan` query exists in supabase-client
- Training plan has `periodization` (phases) and `weekly_template` JSON columns
- No "current week" extraction logic exists yet

## Proposed Approach

### New Dashboard Card: "This Week"

Position: Above Training Load card (below Today's Workout)

**Content:**
- Plan name and current week number (e.g., "Week 3 of 12")
- Current phase name and focus (e.g., "Base Building - Aerobic Endurance")
- Daily workout summary for the week (Mon-Sun with type icons: rest, swim, bike, run)
- Highlight today's slot
- Progress indicator (week X of Y)

**Empty states:**
- No active plan: Don't show the card (or show subtle "Create a plan" prompt)
- Plan paused: Show "Plan paused" with resume button

### Data Flow

1. Dashboard fetches active training plan via `getActiveTrainingPlan`
2. Calculate current week number from `start_date` and today's date
3. Look up current phase from `periodization` JSON
4. Look up daily structure from `weekly_template` JSON
5. Render compact week view

## Files Likely Involved

- `apps/mobile/app/(tabs)/index.tsx` (add card to dashboard)
- `apps/mobile/components/dashboard/WeekOverviewCard.tsx` (new component)
- `apps/mobile/hooks/useTrainingPlan.ts` (may need current-week helper)
- `packages/supabase-client/src/queries/training-plans.ts` (may need getCurrentWeek helper)

## Out of Scope

- Week-by-week plan editing
- Workout detail drill-down from dashboard
- Historical week review
